# Twix
Swift command line tool for replacing the first 5 lines a Swift file with one comment line = '//'.

Turn this:

```swift
//
//  Sample.swift
//  Sample
//
//  Created by Garric G. Nahapetian on 5/13/17.
//  Copyright © 2017 SwiftCoders. All rights reserved.
//

```

Into this:

```swift
//
//  Copyright © 2017 SwiftCoders. All rights reserved.
//
````
